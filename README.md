# Конфигурационные файлы nginx

Конфигурации nginx и Apache для домашнего задания.

## Содержимое

- `load-balancer` - конфигурация nginx с балансировкой
- `site-8080.conf` - Apache на порту 8080
- `site-8081.conf` - Apache на порту 8081  
- `site-8082.conf` - Apache на порту 8082
- `apache-ports.conf` - настройка портов Apache
